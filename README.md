# TASK MANAGEMENT SYSTEM

A Laravel API x React project

## Description

A simple task management system that uses Laravel for backend and ReactJS for frontend

## Getting Started

For ease of use, environment config file (.env) is included in the repository.

### Dependencies

* Composer
* PHP 7.4 +
* npm
### Installing

Clone the project through git
```
git clone https://github.com/jeffrey252/taskmanagementsystem.git
```

### Setting up the site

On a terminal inside the project folder
```
composer install
npm install
npm run dev
```
```
php artisan config:cache
php artisan migrate
```
